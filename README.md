Personalized Real Estate Recommendation Engine

Project Overview:

HomeMatch is an intelligent real estate assistant that transforms traditional property listings into personalized narratives, matching listing to the buyer's lifestyle, needs and preferences using: 

Large Language Models(LLMs) - OpenAI GPT-3.5
Semantic Vector Search - ChromaDB
Text Embeddings - OpenAI Embeddings
Personalised Augumented Listing Descriptions

Features:
1. Synthetic Real Estate Listing Generation
2. Vector Database storage
3. Buyer Preference Collections
4. Sematic Search of Listings
5. Personalised Listing Augmentation

Requirements:

Install with:
pip install -r requirements.txt

Vocareum API setup:
import openai
openai.api_key = "voc-XXX"
